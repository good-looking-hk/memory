技术实现：
  • 1.以市面主流的Spring Boot 作为后台框架，搭建完善的后台服务接口
  • 2.后端数据库采用新型技术JPA（基于Hibernate的一种技术），以实现更简单高效的数据库操作
  • 3.以市面上主流的前后端分离作为规范，前端使用Layui作为前端框架
  • 4.采用主流的缓存工具Redis作为缓存
  • 5.页面层摒弃传统的JSP，而采用新型技术 Thymeleaf作为视图展现（由于大部分采用前后端分离，这个用得并不多）
  • 6.整体技术栈为：spring boot + jpa + mysql + redis + layui + html/js/css
项目说明：
  • 1.本项目历时2个月有余，期间查阅/学习了大量书籍、大量官方文档、大量博客以及大量别人的源代码，学到了很多很多东西
